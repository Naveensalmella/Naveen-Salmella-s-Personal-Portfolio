# Personal Portfolio

A responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This project showcases skills, education, projects, certifications, and contact information with a clean, modern, and interactive user interface.

## Features

* Responsive design for all devices
* Modern and interactive UI
* Smooth scrolling navigation
* Hero section
* About section
* Education timeline
* Technical skills with progress bars
* Soft skills with progress bars
* Projects showcase
* Certifications section
* Contact section
* Scroll animations and hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts
